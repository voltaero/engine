# RFC 1000: Ergonomic use of the Event System

## Summary

Replace the current declarative macro system and manual `Event` trait impls with proc macros and auto-registration.

## Design

### Defining Events

```rust
#[derive(Clone, Debug, Event)]
#[event(namespace = "core", name = "auth_event")]
pub struct AuthEvent {
    pub cancelled: bool,
    pub id: Identifier,
    pub uid: String,
    pub challenge: String,
    pub db: Db,
    pub output: Arc<RwLock<bool>>,
}
// Event trait is fully auto-implemented
```

### Stateless Self-Registering Handlers

```rust
#[event_handler(event = AuthEvent, namespace = "core", name = "auth_event")] // constricts fn to fn(t:eventType)
fn auth_handler(event: &mut AuthEvent) {
    *event.output.write().unwrap() = true;
}
// Generates struct AuthHandler + EventHandler impl + inventory::submit!
// Discovered automatically at startup via api.event_bus.register_all_handlers()
```

### Stateful Handlers (with context)

```rust
#[event_handler(AdminAuthEvent)]
fn admin_auth_handler(event: &mut AdminAuthEvent, token: &str) {
    if token == event.payload {
        *event.output.write().unwrap() = true;
    }
}
// Generates struct AdminAuthHandler with with_ctx() builder
// Registered manually: api.event_bus.on("core", "admin_auth_event", AdminAuthHandler.with_ctx(token))
```
### Cancellation

Events opt into cancellation with the `cancellable` attribute. The `EventBus` skips remaining handlers when an event is cancelled.

```rust
#[derive(Clone, Debug, Event)]
#[event(namespace = "core", name = "auth_event", cancellable)]
pub struct AuthEvent { ... }
```

Handlers can opt in to still receive cancelled events:

```rust
#[event_handler(event = AuthEvent, namespace = "core", name = "auth_event", receive_cancelled = true)]
fn audit_handler(event: &mut AuthEvent) { ... }
```

Updated dispatch loop:

```rust
pub fn fire<T: Event>(&self, event: &mut T) {
    for handler in self.event_handler_registry.get(&event.get_id()) {
        if event.is_cancelled() && !handler.receive_cancelled() { continue; }
        handler.handle(event);
    }
}
```

### Auto-Discovery

Uses [`inventory`](https://docs.rs/inventory). Stateless `#[event_handler]` handlers are submitted at link time; stateful handlers are registered manually.

```rust
inventory::collect!(HandlerRegistration);

// discovers all stateless handlers at startup
api.event_bus.register_all_handlers();
```


### Startup

```rust
// auto-discovers all #[event_handler(..., namespace, name)] handlers
api.event_bus.register_all_handlers();
```
